# EPANET

## Input File Format

- Option
    - hydraulic simulation
        - components
            - pump
                - Global pump efficiency (percent)
        - flow unit: CMH
        - convergence
            - maximum trials
            - ACCURACY
            - maximum head loss error
            - maximum flow change
    - Water quality
        - TOLERANCE

- OPTIONS
    - hydraulic simulation
        - choose
            - UNITS
                - LPS
                - LPM
                - MLD
                - CMH
                - CMD
            - HEADLOSS
                - H-W: Hazen-Williams
                - D-W: Darcy-Weisbach
                - C-M: Chezy-Manning
            - PATTERN
            - DEMAND MULTIPLIER
            - EMITTER EXPONENT
            - DEMAND MODEL
                - DDA
                - PDA
                    - MINIMUM PRESSURE
                    - REQUIRED PRESSURE
                    - PRESSURE EXPONENT
        - solve
            - run
                - TRIALS
            - convergence
                - ACCURACY
                - HEADERROR
                - FLOWCHANGE
            - fail
                - UNBALANCED
                    - STOP
                    - CONTINUE
                    - CONTINUE n
        - result
            - HYDRAULICS
                - SAVE
                - USE
    - water quality
        - QUALITY
            - NONE
            - CHEMICAL
            - AGE
            - TRACE
        - VISCOSITY
        - DIFFUSIVITY
        - SPECIFIC GRAVITY
        - TOLERANCE
